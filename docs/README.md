# Documentation

This folder contains the documentary backbone of the I-SCEET project.

## Structure

### do178/
Contains DO-178C-like management and planning documents:
- PSAC.md
- SDP.md
- SVP.md
- SQAP.md
- SCMP.md

### engineering/
Contains technical engineering inputs and strategy:
- PDS.md
- SYSTEM_ARCHITECTURE.md
- HARDWARE_CONTEXT.md
- TRACEABILITY_STRATEGY.md

### report/
Contains academic and internship-related report material:
- I-SCEET.tex
- JOURNAL_DE_BORD.md
- BILAN_PERIODIQUE.md

### templates/
Contains reusable templates for generated or manually written artifacts:
- HLR_TEMPLATE.md
- LLR_TEMPLATE.md
- TEST_TEMPLATE.md
- TRACEABILITY_TEMPLATE.md

### diagrams/
Contains project figures and architecture diagrams.

## Purpose

The objective of this folder is to separate project documentation from source generation logic and to keep all engineering artifacts structured and maintainable.
